# QA Chatbot using n8n

A simple AI-powered QA chatbot workflow developed using n8n.

## Features

- AI-powered question answering workflow built using n8n
- Focused on answering n8n automation-related queries
- Demonstrates node connections and automation architecture
- Beginner-level agentic AI workflow experiment
- Easily extendable for customized chatbot queries that match your requirements

## Workflow

Chat Trigger → AI Processing → Response

## Workflow Import

1. Open n8n
2. Import the workflow JSON from `workflows/`
3. Configure credentials
4. Activate workflow
